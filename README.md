# Java Review Codes

一些Java课程示例代码，用于辅助理解Java课程。参考教材为Java程序设计教程（第三版），雍郡海著

![textbook](https://img.shields.io/badge/textbook-w-yellow.svg)

## 目录
```
JavaReviewCodes/                           
├── database/                   # 数据库示例
├── enumClass/                  # 枚举类示例
├── exceptionHandle/            # 异常处理示例
├── generic/                    # 泛型示例
├── multithreas/                # 多线程相关
│   ├── synchronizedAndLock/    # 同步与锁机制示例
│   ├── thread/                 # 线程示例
│   └── threadCommunication/    # 线程间通信示例
├── network/                    # 网络相关
│   ├── tcp/                    # tcp示例
│   └── udp/                    # udp示例
└── stream/                     # 流相关
    ├── fileOperation/          # 文件操作示例
    ├── serialization/          # 序列化示例
    └── systemStream/           # 标准流示例
```
